# Casa Associates Website

## 1. Project Description
Casa Associates is a Hyderabad-based consultancy service that helps clients obtain building permissions and government approvals (GHMC, HMDA, TS-bPASS). The website is a lead-generation platform designed to drive WhatsApp contacts, phone calls, and form submissions. Target audience: property owners, builders, and developers in Hyderabad, Telangana.

## 2. Page Structure
- `/` — Single-page website with smooth scroll sections:
  - Hero
  - Services Overview
  - About Casa Associates
  - Detailed Services
  - Process Steps (How It Works)
  - Documents Required
  - Why Choose Us
  - FAQ
  - Contact Section (form + map)
  - Footer
